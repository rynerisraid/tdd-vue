# mini-vuex

Vue全局状态：

* Vuex 是全局状态管理模式库
* 统一的状态存储
    * store 把数据存起来
    * state 应该是响应式，数据发生变化，组件需要实时渲染
* 可预测的变更
    * store.commit()
    * store.dispatch()
        * 用户配置的mutations，mutations[type]
* 体验Vuex的使用

```bash
yarn create vite
yarn add vuex
```
